# Data Structures & Algorithms Applications

A collection of real-world applications demonstrating the practical use of advanced Data Structures and Algorithms. The project focuses on efficient task management, academic ranking, and inventory processing using optimized data structures.

## Features

### 1. Smart Task Management System
- Priority-based task scheduling using **Max Heap**
- Fast task lookup using **Hash Map**
- Undo functionality using **Stack**
- Efficient task insertion and deletion

### 2. Exam Ranking System
- Finds top-performing students using **Min Heap**
- Fast student record management using **Hash Map**
- Supports rank generation and score updates

### 3. Inventory & Billing System
- Order processing using **Queue (FIFO)**
- Item cancellation using **Stack**
- Low-stock alerts using **Min Heap**
- Inventory tracking and billing management

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Data-Structures-Algorithms-Applications.git
```

2. Go to the project directory

```bash
cd Data-Structures-Algorithms-Applications
```

3. Compile the program

```bash
g++ filename.cpp -o app
```

4. Run the executable

```bash
./app
```

> On Windows:

```bash
app.exe
```
